# Flutter eCommerce
[![](https://img.shields.io/badge/Flutter-0553B1?style=for-the-badge&logo=flutter&logoColor=white)](https://flutter.dev/)

This repository centralizes the development of an eCommerce App in Flutter, serving as a practical project that documents the entire journey of creating an app, from developing features and improvements to production-ready delivery.

The project is developed following best practices and community standards proposed by **Andrea Bizzotto (Google Developer Expert)**, through courses available at [Code with Andrea](https://codewithandrea.com/courses/).

## Stack

| Category | Technologies |
|-----------|------------|
| **Language** | Dart (Null Safety) |
| **Architecture** | Clean Architecture |
| **State Management** | Riverpod |
| **Navigation** | GoRouter, Navigator 2.0 |
| **Local Persistence** | Sembast (NoSQL)  |
| **Firebase** | Authentication, Cloud Firestore, Firebase Storage, Local Emulator, Cloud Functions, Remote Config |
| **Integrations** | Stripe (Payments), Algolia (Full-Text Search) |
| **CI/CD** | Codemagic, GitHub Actions, Fastlane |
| **Error Monitoring** | Sentry, Firebase Crashlytics  |
| **Analytics** | Mixpanel, Firebase Analytics  |
| **Code Push** | Shorebird  |
| **Testing** | Mocking, Unit Tests, Widget Tests, Integration Tests, Golden Image Tests, Robot Tests |
| **Distribution**| App Store, Play Store |
